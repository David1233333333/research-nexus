# 🔬 ResearchNexus AI | 學術研究助理 (Academic Research Assistant)

---

> [!TIP]
> ### 💡 專案簡介 | Introduction
> **ResearchNexus AI** 是一個專為學術研究打造的 AI 思路建構工具。  
> 透過串接 Google 的大型語言模型（LLM），協助研究者從不同專業視角審視研究課題。
>
> ResearchNexus AI is a specialized AI framework generator designed for academic research.  
> It leverages Google large language models via the Gemini API to help researchers evaluate topics from diverse professional perspectives.

## 🚀 核心亮點 | Key Features

* **🎓 核心分析 | Senior Professor**  
  提供完整背景分析、研究假設與實驗方法建議。  
  Provides in-depth background analysis, research hypotheses, and experimental methodology.

* **🌱 研究破冰 | Mentor**  
  將複雜學術構思拆解為易理解的入門方向與技術切入點。  
  Simplifies complex academic concepts into accessible entry points and technical directions.

* **⚖️ 壓力測試 | Reviewer**  
  扮演嚴苛審稿人，指出潛在邏輯與實驗設計缺陷。  
  Acts as a rigorous reviewer to identify logical flaws and experimental design weaknesses.

## 📖 使用指南 | Getting Started

### 方法一：線上使用（推薦） | Option 1: Use Online

1. 前往線上版本：  
   Visit the live site:  
   👉 https://david1233333333.github.io/research-nexus/

2. 若尚未申請 API Key，點擊頁面上的 **Get Key**。  
   If you do not yet have an API Key, click **Get Key** on the page.

3. 在頂部輸入框貼入您的 API Key。  
   Paste your API Key into the top input field.

4. 輸入研究主題並點擊 **Generate Research Report**。  
   Enter your research topic and click **Generate Research Report** to start analysis.

---

### 方法二：本地使用 | Option 2: Run Locally

1. Clone 此專案。  
   Clone the repository.

2. 使用瀏覽器開啟 `index.html`。  
   Open `index.html` in your browser.

3. 點擊 **Get Key** 取得 API Key，並開始使用。  
   Click **Get Key**, paste your key, and start generating analysis.
