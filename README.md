# 🔬 ResearchNexus AI | 學術研究助理 (Academic Research Assistant)



---

> [!TIP]
> ### 💡 專案簡介 | Introduction
> **ResearchNexus AI** 是一個專為學術研究打造的 AI 思路建構工具。透過串接 Google 最新一代的 **Gemma 3** 模型，幫助研究者從不同視角審視研究課題。
> 
> A specialized AI framework generator for academic research. Leveraging Google's next-gen **Gemma 3** model to help researchers evaluate topics from diverse professional perspectives.

### 🚀 核心亮點 | Key Features

* **🎓 核心分析 | Senior Professor**
  提供完整的背景分析、研究假設與實驗方法建議。
  Provides in-depth background analysis, research hypotheses, and experimental methodology.

* **🌱 研究破冰 | Mentor**
  將複雜的學術構思拆解為平易近人的白話文與入門點。
  Simplifies complex academic concepts into layman terms and accessible entry points.

* **⚖️ 壓力測試 | Reviewer**
  扮演嚴苛審稿人，精準找出邏輯漏洞與實驗缺陷。
  Acts as a rigorous peer reviewer to identify logical flaws and design defects.

### 📖 使用指南 | Getting Started

> [!IMPORTANT]
> **隱私聲明 | Privacy Note**
> 本工具為純前端應用，API Key 僅存於瀏覽器暫存，絕不進行後端存儲。
> This is a client-side application. Your API Key is used only for browser requests and is never stored on a server.

1. **取得金鑰 | Get API Key**
   前往 [Google AI Studio](https://aistudio.google.com/app/api-keys) 免費申請。
   Visit [Google AI Studio](https://aistudio.google.com/app/api-keys) to get your free Gemini API Key.

2. **開啟工具 | Open the Tool**
   直接使用瀏覽器開啟 `index.html` 檔案。
   Open the `index.html` file directly in any modern web browser.

3. **輸入金鑰 | Enter Key**
   在頂部輸入框貼入您的 API Key。
   Paste your API Key into the designated input field at the top.

4. **開始分析 | Run Analysis**
   輸入研究主題並點擊 **「啟動學術分析報告」**。
   Enter your research topic and click **"Generate Research Report"**.
