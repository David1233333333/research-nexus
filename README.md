# 🔬 ResearchNexus AI | 學術研究助理 (Academic Research Assistant)

---

> [!TIP]
> ### 💡 專案簡介 | Introduction
> **ResearchNexus AI** 是一個專為學術研究打造的 AI 思路建構工具。  
> 透過串接 Google 的大型語言模型（LLM），幫助研究者從不同專業視角審視研究課題。
>
> A specialized AI framework generator for academic research.  
> Leveraging Google large language models via the Gemini API to help researchers evaluate topics from diverse professional perspectives.

## 🚀 核心亮點 | Key Features

* **🎓 核心分析 | Senior Professor**  
  提供完整的背景分析、研究假設與實驗方法建議。  
  Provides in-depth background analysis, research hypotheses, and experimental methodology.

* **🌱 研究破冰 | Mentor**  
  將複雜的學術構思拆解為平易近人的白話文與入門點。  
  Simplifies complex academic concepts into layman terms and accessible entry points.

* **⚖️ 壓力測試 | Reviewer**  
  扮演嚴苛審稿人，精準找出邏輯漏洞與實驗缺陷。  
  Acts as a rigorous peer reviewer to identify logical flaws and design defects.

## 📖 使用指南 | Getting Started

1. **開啟工具 | Open the Tool**  
   使用瀏覽器開啟 `index.html`。  
   Open the `index.html` file in any modern web browser.

2. **輸入金鑰 | Enter Key**  
   在頂部輸入框貼入您的 API Key。  
   Paste your Gemini API Key into the input field at the top.

3. **開始分析 | Run Analysis**  
   輸入研究主題並點擊 **Generate Research Report**。  
   Enter your research topic and click **Generate Research Report** to start the analysis.
